# Python Programming Lab Experiments

This repository contains Python programs created as part of the **Python Programming Laboratory coursework**.  
The programs demonstrate basic Python concepts such as **data types, operators, control flow, loops, strings, lists, tuples, and dictionaries**.

All programs are written in a **simple and beginner-friendly way** to help understand Python programming easily.

---

## List of Experiments

1. **Experiment 1:** Demonstrate various data types and operators in Python.  
2. **Experiment 2:** Program to implement control flow and looping statements.  
3. **Experiment 3:** Program to demonstrate string operations.  
4. **Experiment 4:** Program to demonstrate list operations and related functions.  
5. **Experiment 5:** Program to demonstrate tuple operations and related functions.  
6. **Experiment 6:** Program to demonstrate dictionary operations and related functions.

---

## How to Run the Programs

1. Download or clone this repository.
2. Open the project in **VS Code** or any Python IDE.
3. Open the experiment file you want to run (example: `exp1.py`).
4. Run the program using Python.

Example command:

```bash
python exp1.py
